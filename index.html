Personalized Shared Folders

A Personalized Shared Folder is a new form of a shared folder that each collaborator is able to organize the folder structures and file names in his or her own needs.

Integrated Cloud Storage

An Integrated Cloud Storage is a centralized management console for the cloud services such as Dropbox, Google Drive, and Microsoft OneDrive.

Seamless Collaboration.

Seamless Collaboration is a set of techniques allowing an user to collaborate with each other using the Personalized Shared folders and the Integrated Cloud Storage.
